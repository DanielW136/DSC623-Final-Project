# Reliable Rentals - Database Implementation

**DSC623 Database Project**
**Authors:** Daniel Wilcox & Corwin Tilden


This repository contains the implementation of the Reliable Rentals vehicle hire database.

## Database Schema

The database consists of six relations:

| Relation | Primary Key | Description |
|---|---|---|
| Site | siteNumber | Rental outlet locations |
| Models | model | Vehicle model and make information |
| Client | clientNumber | Customer records |
| Staff | staffNumber | Employee records, linked to a site |
| Vehicle | registrationNumber | Vehicle inventory, linked to a model and site |
| HireAgreement | hireNumber | Rental transactions, linked to a client and vehicle |

## Repository Contents

```
schema_and_data.py   - Creates the database, defines the schema, and inserts sample data
transactions.py      - Verifies table contents and runs the five embedded SQL queries
vehicle_hire.db      - SQLite database file generated by schema_and_data.py
Reports/*           - Reports for all project parts
```

## Usage

Run the scripts in order:

```bash
python schema_and_data.py
python transactions.py
```

`schema_and_data.py` creates a fresh `vehicle_hire.db` each time it runs. `transactions.py` connects to that database, prints the contents of all six tables, and then executes the five SQL queries.

## Five Implemented Queries

1. Find all staff members working at the site where a given vehicle is located.
2. Retrieve the address of the client who hired a specific vehicle on a given date.
3. Count the number of vehicles allocated to the site where a specific staff member works.
4. List the make and model of all vehicles under active hire agreements for a given client.
5. Look up the phone number and address of the site associated with a specific client's rental.
